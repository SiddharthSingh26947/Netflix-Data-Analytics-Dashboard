# 🎬 Netflix Data Analytics Dashboard

## 📌 Overview
This project presents an interactive **Netflix Data Analytics Dashboard** designed to analyze content trends, distribution, and viewer insights. The dashboard transforms raw data into meaningful visualizations to support data-driven decision-making.

---

## 🧠 Project Summary
Developed an interactive dashboard to explore Netflix content based on genre, country, release year, and ratings. Applied data cleaning, transformation, and visualization techniques to generate actionable insights. This project highlights strong analytical thinking skill.

---

## 🛠️ Tech Stack
- **Visualization Tool:** Power BI
- **Data Source Format:** CSV

---

## 📂 Data Source
- Dataset sourced from **Kaggle**
- Includes the following attributes:
  - Title  
  - Genre  
  - Release Year  
  - Country  
  - Rating  
  - Duration  

---

## 🚀 Features
- 📊 Interactive dashboard with filters  
- 🎬 Genre-wise content analysis   
- 🌍 Country-wise content distribution  
- ⭐ Ratings-based insights  
- 🔍 Clean and user-friendly interface
